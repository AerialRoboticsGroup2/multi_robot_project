# multi_robot_project

# Multi Robot Cooperative Control

## Features

- Formation control
- Collision avoidance
- Voronoi coverage
- Consensus estimation
- Connectivity maintenance
- Topology control

## Dependencies

- ROS2 Humble
- Python3
- NumPy
- NetworkX

## Build

colcon build
source install/setup.bash

## Run

ros2 launch multi_robot_project multi_robot.launch.py
